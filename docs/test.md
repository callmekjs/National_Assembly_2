## T-1. 데이터 무결성

### T-1-1. DB 연결 및 테이블 존재
**방법**: 터미널에서 psql 또는 Python으로 직접 확인  
```bash
python -c "
import psycopg2
conn = psycopg2.connect(host='localhost', port=5433, database='skn_project', user='postgres', password='post1234')
cur = conn.cursor()
cur.execute('SELECT COUNT(*) FROM chunks')
print('chunks:', cur.fetchone()[0])
cur.execute('SELECT COUNT(*) FROM embeddings_e5')
print('embeddings:', cur.fetchone()[0])
conn.close()
"
```
**PASS 기준**:
- chunks 테이블 15,000건 이상
- embeddings_e5 테이블 chunks와 동일 건수 (±100 이내)
- 연결 오류 없음

---

### T-1-2. 메타데이터 품질
**방법**: 아래 쿼리 실행
```sql
SELECT
  COUNT(*) AS total,
  SUM(CASE WHEN metadata->>'speaker' IS NULL OR metadata->>'speaker' = '' THEN 1 ELSE 0 END) AS speaker_null,
  SUM(CASE WHEN metadata->>'date' IS NULL OR metadata->>'date' = '' THEN 1 ELSE 0 END) AS date_null,
  SUM(CASE WHEN metadata->>'committee' IS NULL OR metadata->>'committee' = '' THEN 1 ELSE 0 END) AS committee_null
FROM chunks;
```
**PASS 기준**:
- speaker 누락률 < 10%
- date 누락률 < 5%
- committee 누락률 < 5%

---

### T-1-3. 임베딩 고아 레코드
**방법**:
```sql
SELECT COUNT(*) FROM embeddings_e5 e
LEFT JOIN chunks c ON e.chunk_id = c.chunk_id
WHERE c.chunk_id IS NULL;
```
**PASS 기준**: 0건 (임베딩이 있는데 청크가 없는 경우 없어야 함)

## T-2. 검색 품질

### T-2-2. Fusion + Neural Reranker 기본 활성화 확인
**방법**: 앱 재시작 + F5 후 사이드바 고급 검색 전략 확인

## T-4. 생성 품질

### T-4-1. 인용 번호 정합성
**방법**: 아무 질문 후 답변 본문의 [n]과 참고자료 테이블 번호 대조  
질문: `"조태열 장관의 한미동맹 발언 정리해줘"`  
**PASS 기준**:
- 본문 [1] → 참고자료 1번 발언자 일치
- 본문 [2] → 참고자료 2번 발언자 일치
- 본문에 없는 번호가 참고자료에 노출되지 않음

---

### T-4-2. Grounding NONE 경고 표시
**방법**: OOD 질문 후 답변 확인  
질문: `"섹스하고 싶다"`  
**PASS 기준**:
- ⚠️ 또는 ❌ 경고 문구 노출
- 참고자료 테이블 없음

---

### T-4-3. 멀티턴 — 대명사 해석
**방법**: 두 질문을 연속 입력 (같은 대화창)  
1턴: `"조태열 장관 대북정책 발언 알려줘"`  
2턴: `"그 사람이 한미동맹에 대해선?"`  
**PASS 기준**:
- 2턴 답변에 조태열 발언 인용
- 2턴 답변에서 "그 사람"을 다른 인물로 해석하지 않음

---

### T-4-4. 멀티턴 — OOD 오염 방지
**방법**: 세 질문 연속 입력  
1턴: `"대북정책 핵심 쟁점은?"`  
2턴: `"외교부에서 마인크래프트 어떻게 생각해?"`  
3턴: `"조태열 장관 한미동맹 발언"`  
**PASS 기준**:
- 3턴 답변에 2턴의 OOD 경고 문구 복붙 없음
- 3턴이 정상 검색 결과로 시작

---

### T-4-5. 스트리밍 응답 정상 동작
**방법**: 질문 입력 후 화면 관찰  
질문: `"남북교류협력 관련 논의 알려줘"`  
**PASS 기준**:
- 텍스트가 타이핑되듯 순차 출력
- 스트리밍 중 화면 깨짐 없음
- 완료 후 참고자료 블록 정상 노출

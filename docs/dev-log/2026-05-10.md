- **`service/rag/generation/generator.py`**: 데모 경로도 동일한 한 줄 인용·컨텍스트 형식(원시 필드 제거).

### 프롬프트 (`service/llm/prompt_templates.py`) 누적
- 우선순위: (1) 출처 한 줄 포맷 → (3) 인라인 각주 → (2) 결론/한계 표현 → (4) 날짜.
- 출처 줄: `[n] (YYYY-MM-DD) 발언자: "요약"` · 원시 ID/필드명 금지 · 인용은 **문장 첫머리부터** · 줄일 때 **끝만** `...`.
- 발언자: 메타의 실제 호칭 사용, **전부 「위원장」으로 통일 금지**.
- **본문 구조**: `## 핵심 결론` / `## 세부 근거` / `## 한계` 만 사용, 섹션 사이 빈 줄.
- **날짜**: 질문 기준일 대비 365일 이상·**회의일이 2025-01-01 이후**이면 줄 끝 `(날짜 확인 필요)` 및 말미 한 줄 안내.
- **질문 주체 일치(강화)**: 질문이 **특정 1인 견해**만 묻는 경우 `## 세부 근거` **모든 불릿**은 그 사람만 주어 · **다른 위원·다른 의원·(별도 참고) 타인 불릿 금지** · **○○ 정부만으로 개인 발언 대체 금지** · 근거 부족 시 `## 한계`에만 기술. 질문에 비교·대조가 **명시**된 경우만 예외.
- `build_user_prompt`: 질문 기준일, 단일인 질문 시 세부 근거 규칙 재강조.
- (3) 각주: 특정 1인 질문 시 **무관 청크에 각주 오인 연결 금지**.

## 2026-05-10 오늘 한 일 — OpenAI로 답변 생성 연동 (LangGraph `Generate`)

- **`service/llm/llm_client.py`**: `.env`에 **`OPENAI_API_KEY`**가 있고 `FORCE_LOCAL_LLM`이 아니면 **`OPENAI_BASE_URL`(기본 `https://api.openai.com/v1`)의 Chat Completions**로 생성(`requests`, `OPENAI_MODEL` 기본 `gpt-4o-mini`). 실패 시 기존 **로컬 HF(LoRA)** 로 폴백. **`OPENAI_ONLY=1`**이면 API 실패 시 로컬로 안 내림. **`FORCE_LOCAL_LLM=1`**이면 항상 로컬만.

- **`.env.example`**: OpenAI 블록 중복 제거·`OPENAI_MODEL`·`FORCE_LOCAL_LLM`·`OPENAI_ONLY` 주석 추가.
- **`README.md`**, **`app.py` 홈**: Streamlit/LangGraph 답변이 **키 있으면 OpenAI 우선**이라는 설명으로 갱신.
- 코드 반영 후 **Streamlit 재시작**하면 `Generate` 노드에서 API 경로가 타는지 확인 가능(터미널에 `[LLM] backend=openai model=...` 로그).

- `Retriever.search(use_mmr=True, mmr_lambda=0.7)`

### Score Normalization (`service/rag/retrieval/score_norm.py`)
- 벡터 유사도·어휘 중복·키워드 부스트 점수를 min-max 정규화 후 가중 앙상블

- `Retriever.search(use_score_norm=True)`

### Multi-reranker Ensemble (`service/rag/retrieval/ensemble_reranker.py`)
- Neural + LLM Reranker를 독립 실행 후 RRF(Reciprocal Rank Fusion)로 순위 합산
- 한 쪽 실패 시 성공한 쪽 결과만 사용 (폴백 안전)
- `Retriever.search(use_ensemble_reranker=True)`

### recall@3 자동 출력 (`service/rag/eval/recall_eval.py`)
- 검색마다 4가지 휴리스틱 지표 자동 계산: keyword_hit_rate / score_gap / diversity / avg_similarity

- `Retriever.search(use_compression=True)`

### Step-back Prompting (`service/rag/retrieval/step_back.py`)
- 구체 질문 → LLM이 추상 질문 생성 → 두 결과 RRF 합산
- 테스트: "통일부 장관 관련 주요 질의?" → "통일부의 정책 방향에 관한 주요 논의?" 변환
- 기존 미발굴 문서 2건 추가(`20240813`, `20250226`)
- `Retriever.search(use_step_back=True)`

### Fusion Retrieval (`service/rag/retrieval/fusion.py`)
- 벡터 검색 결과를 BM25 스코어로 재정렬 후 RRF 통합 (2개 순위 리스트)
- 경량 BM25: k1=1.5, b=0.75, IDF 상수 처리 (단일 문서 기준)
- 테스트: 기존 hybrid score 대비 다른 문서 발굴(`20241216` 신규 등장)
- `Retriever.search(use_fusion=True)`

### HyDE (`service/rag/retrieval/hyde.py`)
- **이전**: 질문 텍스트를 그대로 임베딩 → 질문과 문서 표현 격차 존재
- **개선**: 질문 → GPT-4o-mini가 가상 답변 생성(~300자) → 그 답변을 임베딩해서 검색
- 재랭킹은 원본 질문 기준으로 수행해 relevance 보정
- 실패 시 원본 질문으로 자동 폴백
- `Retriever.search(use_hyde=True)` / LangGraph `meta["use_hyde"]` 로 제어
- 테스트 결과: `통일부 장관` 질의에서 기존 검색과 완전히 다른 문서 탐색 (겹침 0개)

### Multi-query Retrieval (`service/rag/retrieval/multi_query.py`)

### Multi-query Retrieval (`service/rag/retrieval/multi_query.py`)
- **이전**: 질문 1개 → 단일 검색
- **개선**: 질문 1개 → LLM(GPT-4o-mini)이 변형 질문 3개 생성 → 총 4회 검색 → RRF(Reciprocal Rank Fusion)로 통합
- `generate_queries()`: OpenAI로 변형 질문 생성, JSON 파싱 실패 시 원본만 사용(폴백)
- `rrf_merge()`: `score = Σ 1/(60 + rank)` — 여러 결과 리스트를 chunk_id 기준으로 합산
- `multi_query_search()`: Retriever를 래핑해 멀티쿼리 전체 파이프라인 실행
- `Retriever.search(use_multi_query=True, multi_query_variants=3)` 로 활성화
- LangGraph `retrieve_pg.py` — `meta["use_multi_query"]`로 Streamlit UI에서도 제어 가능
- 테스트 결과: 기존 검색 대비 다른 문서 1건 추가 발굴 (`20240717_52128` 등장)
